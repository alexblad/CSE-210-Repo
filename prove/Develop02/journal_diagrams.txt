##The Class diagrams for the journal program 
##The program will have # classes in addition to the program class 

Class: Journal 
Attributes: 
* _menuOptions : List<option>
* _entries : List<Entry>

Methods: 
* DisplayOptions() : void 
    -displays menu options
* NewEntry(newEntry : Entry) : void 
    -create a new entry
* DisplayEntries() : void 
    -displays all the entries in the journal
*SaveToFile(file : string) : void
    -saves the file
*LoadFromFile(file : string) : void
    -loads file 
* quit() : void 
    -quits the journal



Class: Entry 
Attributes: 
* _date : string (or datetime?)
* _prompt : string 
* _entryText : string 
* _location : string

Methods: 
* DisplayEntry() : void 
* EntryResponse() : void

Class: Prompt Generator
Attributes: 
* _prompts : List<string> 

Methods: 
* GetRandomPrompt() : string
